# Waddle Away

A small 2D endless runner built with Godot where you play as a penguin, avoid obstacles and survive for as long as possible.

Built as a learning project focused on finishing and shipping a complete game rather than leaving another prototype unfinished. The goal was to take a project through the full development cycle, from implementation and playtesting to a polished browser release.

**Play in your browser:**  
https://umerkniazi.itch.io/waddle-away

*Mobile devices are supported with dedicated touch controls.*

## Development

This project focused on:

- Completing a game from start to finish
- Learning Godot's scene-based workflow and GDScript
- Building and deploying a browser version with itch.io
- Keeping the scope intentionally small while polishing the gameplay
