# Waddle Away

A small 2D endless runner built with Godot where you play as a penguin, avoid obstacles, and survive for as long as possible.

I built Waddle Away to practice taking a game from idea to release instead of leaving another prototype unfinished. Rather than aiming for a large or complex project, the goal was to complete the full development cycle, from implementation and playtesting to a polished browser release.

> **Play in your browser:** https://umerkniazi.itch.io/waddle-away
>
> *Mobile devices are supported with dedicated touch controls.*

> **Tip:** Add a gameplay GIF or screenshot here. For game projects, visuals make a much stronger first impression than text.

## Development

This project focused on:

- Completing a game from start to finish
- Learning Godot's scene-based workflow and GDScript
- Building and deploying a browser version with itch.io
- Improving gameplay polish while keeping the scope intentionally small

## Built With

- **Engine:** Godot
- **Language:** GDScript
- **Distribution:** itch.io
